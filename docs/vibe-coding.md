# PANDAI Vibe Coding Protocol

## Mindset
1. **AI-First**: Selalu optimalkan instruksi untuk koordinasi antara manusia dan AI assistant.
2. **Speed with Quality**: Fokus pada iterasi cepat namun tetap menjaga struktur kode yang bersih dan terdokumentasi.
3. **Modular Design**: Setiap komponen harus independen dan mudah diuji.

## Gaya Penulisan Kode
- **Naming Convention**: Gunakan `camelCase` untuk Javascript/React dan `snake_case` untuk Python (backend/Neuro-Client).
- **Commit Messages**: Gunakan [Conventional Commits](https://www.conventionalcommits.org/).
- **Comments**: Tulis komentar yang menjelaskan "Mengapa" (Why), bukan "Apa" (What).

## Workflow (AIDD - AI Driven Development)
1. Definisikan spesifikasi di folder `docs/`.
2. Gunakan prompt yang terstruktur untuk inisiasi modular.
3. Validasi setiap fitur dengan tool testing yang tersedia sebelum digabung ke `main`.
